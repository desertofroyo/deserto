Warm content card with optional media and hover lift.

```jsx
<Card media={<img src="bowl.jpg" />} interactive>
  <h3>Mango Tango Bowl</h3>
  <p>Mango froyo, granola, fresh berries.</p>
  <Button variant="accent">Add — $7.50</Button>
</Card>
```

`elevation`: none | sm | md | lg. Set `interactive` for clickable cards (lift + shadow on hover).
