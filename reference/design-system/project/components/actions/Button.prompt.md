Pill-shaped Deserto button — use for any primary or secondary action.

```jsx
<Button variant="primary">Order now</Button>
<Button variant="accent" iconRight={<ArrowRight size={18} />}>Build your bowl</Button>
<Button variant="outline" size="sm">View menu</Button>
```

Variants: `primary` (wine, default), `accent` (orange CTA with glow), `secondary` (soft wine wash), `outline`, `ghost`. Sizes: `sm | md | lg`. Props: `fullWidth`, `iconLeft`, `iconRight`, `disabled`. Hover darkens one step; press scales to 0.97.
